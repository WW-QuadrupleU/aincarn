# Aincarn Lab データ保管場所

ここは、AI比較の実測結果を一時保管する場所です。
各AIの出力をコピペして、あとで公開用データへ整理します。

## 使い方

1. `raw/` の該当カテゴリファイルに、各AIの回答をそのまま貼る
2. 比較日、モデル名、利用環境、所要時間、料金目安を分かる範囲で記録する
3. `processed/` に公開用の要約・評価を整理する
4. 公開する内容を `lib/aincarn-lab.ts`、または将来のJSON/MDXへ反映する

## フォルダ

- `raw/`: コピペ用の生ログ
- `processed/`: 公開前に整理した比較結果

## 注意

- 出力全文を公開ページに長く載せすぎない
- 各社AIの出力は、公開時に要約・評価として扱う
- 比較日は必ず残す
- 「今回の条件では」という前提を入れる
- 料金やモデル名は変わるため、確認日を残す
